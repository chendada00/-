# 正则表达式



| 字符    | 描述                   |
| ------- | ---------------------- |
| .       | 匹配任意一个字符       |
| ^ab     | 匹配ab开头的字符串     |
| ab$     | 匹配ab结尾的字符串     |
| ab*     | 匹配ab零次或多次       |
| ab+     | 匹配ab一次或多次       |
| ab?     | 匹配ab零次或一次       |
| ab{n}   | 匹配abn次              |
| ab{n,}  | 匹配ab至少n次          |
| ab{n,m} | 匹配ab至少n次，至多m次 |
| a\|b    | 匹配a或b               |
